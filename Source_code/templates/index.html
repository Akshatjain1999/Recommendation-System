recommendations

MovieRecommendator helps you find movies you will like. Rate movies to build a custom taste profile, then MovieRecommendator recommends other movies for you to watch.

rich data

Learn more about movies with rich data, images, and trailers. Browse movies by community-applied tags, or apply your own tags. Explore the database with expressive search tools.

more ninja

Find movies that are similar to the ones you like. Tune the matching algorithm so that the results are "less violent", "more realistic", or "more ninja".

science

MovieRecommendator is run by Vertos, a research lab at the Lovely Professional University. By using MovieRecommendator, you will help vertos develop new experimental tools and interfaces for data exploration and recommendation. MovieRecommendator is non-commercial, and free of advertisements.
